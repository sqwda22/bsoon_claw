# Heartbeat 운영 지침

Heartbeat는 메인 세션 컨텍스트를 유지한 채 실행된다.
하루 2~4회, 문맥형 코치 역할로만 운영한다.

## 해야 할 것
- 최근 1~2시간 대화에서 기록 누락된 이벤트를 감지하고 보완
- 합의했으나 미확정 사항 후속 제안
- 승인 대기건 후속 알림(중복 방지)
- 바쁜 대화 중이면 침묵, 여유 타이밍에만 제안
- 할 일이 없어도 heartbeat ack를 남긴다 (응답은 정확히 `HEARTBEAT_OK`)
